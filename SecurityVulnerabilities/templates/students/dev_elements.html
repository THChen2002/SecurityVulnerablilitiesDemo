Elements

Elements 主要用來檢視與調整 DOM 元素以及 Styles CSS 屬性。

DOM 元素相關

  • 編輯 DOM 元素

    • 點選元素兩下，可快速編輯元素內容、屬性
    點選元素→右鍵→Edit as HTML，可如同在編輯器中編輯 HTML 一般的編輯 DOM 元素

  • 快速收合 DOM 結構

    • 點選元素→右鍵→ Expand recursively (展開全部子元素) 點選元素→右鍵→ Collapse children (收合至第一層子元素)

  • 取得 DOM 元素

    • 方法一：在 Elements 選取元素，接著在 Console 輸入 $0，即可獲得該元素。亦可獲得 $1、$2..，代表的是上一次選取的元素、上上一次選取的元素..，以此類推。

    方法二：點選元素→右鍵→Copy→Copy JS path，接著在 Console 貼上即可獲取該元素。

    方法三：存下來當作全域變數，DevTools 預設變數名稱為 temp 加上數字，第一個為 temp1，第二個為 temp2.. 以此類推。

    畫面位移至被選取的元素位置
    點選元素→右鍵→ Scroll into view

Styles 相關

  • 編輯 CSS
    1. 點選欲編輯的 DOM 元素後，於 Styles 中的 element.style 輸入 CSS 屬性，可以直接看到編輯結果
    2. 點選 :hov ，可選取不同狀態
    3. 點選 + 號，可進行偽類、偽元素的編輯
  • 查看所有屬性

    • box model 用以查看元素大小的組成要素(margin、border、padding、content)，這個盒子對於剛學習 CSS 的人來說，非常有幫助哦！
    box model 下方的列出各屬性值，也可點選 show all 查看所有包含繼承的屬性。

  • 查找屬性設定來源

    • 設定 CSS 屬性時，有時因層級設定等關係，畫面沒有如預期的顯示出我們想要設定的樣式，這時很需要知道目前的屬性值來源。以找出 color 屬性設定來源為例，有兩種方式。

    1. 方法一：在 Filter 欄位中查找特定屬性。
    2. 方法二：查找屬性值列表，點選 color →來源箭頭，導向來源

reference : Get started with viewing and changing the DOM、View the properties of DOM objects、開發必備，摸索不完的 Chrome DevTools 小技巧之 Element 篇
